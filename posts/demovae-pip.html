Tue Jun 18 2024 DemoVAE Model Available as Installable Pip Package Hide

I've updated the DemoVAE (demographic-conditioned variational autoencoder) project for you to be able to install the model using pip.

Click here for a demo of the generative model trained with the PNC and BSNIP datasets.

A preprint describing the DemoVAE model and it's use in removing demographic confounds from fMRI FC data can be found here.

Code and tutorials can be found on the GitHub project.
