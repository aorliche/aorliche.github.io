September 5, 2022: Check out the demo running on a Linode cloud instance!

The demo comes preloaded with analysis of 2 small datasets. Info on the datasets used can be found on the ImageNomeR github.

August 30, 2022: Our pre-print has been posted on arXiv!

Latent similarity is a tool used to estimate differences between subject endophenotypes based on target variables. It is especially good in the small-sample-size regime.

August 23, 2022: The LatSim code is now available on GitHub; we are working on a PIP package.

August 8, 2022: We have completed part of the tool as the ImageNomeR project at the UAB Multiomics Hackathon
